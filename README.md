# Chicken Run API

## Installation

```bash
$ yarn install
```

## Running the app

#### Start the database
```bash
$ docker-compose up
```

#### Start the API
```bash
$ yarn start
```

## API Usage

Once started, the API is exposed on `localhost:3000`, visit `/api` to have detail about Chicken run API usage.
